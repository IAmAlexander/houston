What is Houston?

Houston is a central hub for all things your company, from upcoming projects to who's here:

  • Plan projects on a beautiful interactive timeline
  • See who's out this week and next
  • Submit and approve leave requests
  • Browse the company handbook
  • Meant for project managers, CEO's, developers, designers—anyone part of your team's success!

Houston is intelligent and hooks into the favorite tools you already use.

  • Google apps for sign-on and spreadsheet planning
  • Atlassian Confluence for company wikis
  • Harvest for time tracking
  • Google Analytics
  • ... and developed on Ruby on Rails!

Get Started

Getting Houston up and running is a snap. Simply fork this repository, and deploy! We recommend an easy platform like Heroku. You'll have the option to configure Google sign-on and other services once you're up and running.


$ git push heroku 
$ heroku run rake db:setup

To get started locally with Rails:

$ cp config/database.yml.template config/database.yml

(You'll want to replace the defaults with your own database settings.)


$ bundle install
$ bundle exec rake db:setup
$ bundle exec rails s

That's it? Yep, that's it.

We've provided you with some dummy projects to start. To begin working with your own projects, simply follow these steps:

  1. Visit this Google spreadsheet to access the dummy projects template. It'll be read-only, but don't fret.
  2. Copy this spreadsheet to your own organization. Voila, you can edit it now!
  3. Once you're logged into Houston, access "Switch Docs" from the menu, and change the key to your own Google Spreadsheet.
  4. Visit Schedules to import your new spreadsheet.

For full documentation on all the nifty features, check out the Houston wiki.
